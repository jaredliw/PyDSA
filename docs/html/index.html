Welcome to PyDSA’s documentation!¶

PyDSA is a collection of data structures and algorithms implemented in Python. The main aim of this project is to help python learners in understanding DSA and apply them in problem-solving as well as in competitive programming.

Github repository: https://github.com/jaredliw/PyDSA

Requirements and Dependencies¶

Minimum required version for Python is Python 3.7. Python 2 is NOT supported.

For more details, please read requirements.txt.

Installation¶

git clone https://github.com/jaredliw/PyDSA.git

or directly download ZIP.

Afterwards, install all dependencies by running the command:

pip install -r requirements.txt
