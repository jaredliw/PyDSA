Linked List¶

A linear data structure where each element is a separate object.

exception pydsa.data_structures.linked_list.ExceededMaxIterations¶

Bases: RuntimeError

Raised when maximum iterations has been exceeded. This is usually caused by a cycle inside a linked list.

class pydsa.data_structures.linked_list.SinglyLinkedList(iterable=None)¶

Bases: object

A one-way linear data structure where elements are separated and non-contiguous objects that linked by pointers.

Note

SinglyLinkedList supports all methods from built-in list, except indexing / slicing (use traverse() for indexing). Besides that, unlike list, copy() is making a deep copy.

Variables

  • MAX_ITER – Maximum number of iterations, process will be terminated if it has been exceeded.

  • head – Head of linked list.

Raises

ExceededMaxIterations – Raised when maximum iterations has been exceeded to prevent an infinite loop.

index(value, start=0, end=9223372036854775807)¶

Return first index of node with value. The optional arguments start and end are used to limit the search to a particular subsequence of the linked list. The returned index is computed relative to the beginning of the full sequence rather than the start argument.
