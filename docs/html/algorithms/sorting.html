Sorting¶

An algorithm that is used to rearrange a given array according to a comparison operator on the elements.

pydsa.algorithms.sorting.bead_sort(arr, key=<function <lambda>>, reverse=False)¶

Also known as Gravity sort, this algorithm was inspired from natural phenomena and was designed keeping in mind objects(or beads) falling under the influence of gravity.

pydsa.algorithms.sorting.bogobogosort(arr, key=<function <lambda>>, reverse=False, randomized=False)¶

An algorithm that was designed not to succeed before the heat death of the universe on any sizable list. It works by recursively calling itself with smaller and smaller copies of the beginning of the list to see if they are sorted.

pydsa.algorithms.sorting.bogosort(arr, key=<function <lambda>>, reverse=False, randomized=False)¶

An ineffective algorithm based on generate and test paradigm.

pydsa.algorithms.sorting.bozosort(arr, key=<function <lambda>>, reverse=False)¶

random sorting algorithms where the key idea is to swap any two elements of the list randomly and check if the list is sorted.

pydsa.algorithms.sorting.bubble_sort(arr, key=<function <lambda>>, reverse=False)¶

An algorithm that works by repeatedly swapping the adjacent elements if they are in wrong order.

pydsa.algorithms.sorting.bucket_sort(arr, key=<function <lambda>>, reverse=False, sorting_algo=<function insertion_sort>)¶

Bucket sort is a sorting algorithm that works by distributing the elements of an array into a number of buckets. Each bucket is then sorted individually, either using a different sorting algorithm, or by recursively applying the bucket sorting algorithm.

pydsa.algorithms.sorting.cocktail_sort(arr, key=<function <lambda>>, reverse=False)¶

A variation of Bubble Sort. It traverses through a given array in both directions alternatively.

pydsa.algorithms.sorting.comb_sort(arr, key=<function <lambda>>, reverse=False)¶

Improve on Bubble Sort by using gap of size more than 1.

pydsa.algorithms.sorting.counting_sort(arr, reverse=False)¶

Counting sort is a sorting technique based on keys between a specific range. It works by counting the number of objects having distinct key values.

pydsa.algorithms.sorting.gnome_sort(arr, key=<function <lambda>>, reverse=False)¶

a sorting algorithm which is similar to insertion sort in that it works with one item at a time but gets the item to the proper place by a series of swaps, similar to a bubble sort.

pydsa.algorithms.sorting.insertion_sort(arr, key=<function <lambda>>, reverse=False)¶

An algorithm that works by splitting the array into a sorted and an unsorted part and values from the unsorted part are picked and placed at the correct position in the sorted part.

pydsa.algorithms.sorting.is_sorted(arr, key=<function <lambda>>, reverse=False)¶

Check whether the list is sorted.

pydsa.algorithms.sorting.merge_sort(arr, key=<function <lambda>>, reverse=False)¶

A divide-and-conquer algorithm that divides the input array into two halves, calls itself for the two halves, and then merges the two sorted halves.

pydsa.algorithms.sorting.odd_even_sort(arr, key=<function <lambda>>, reverse=False)¶

A variation of Bubble Sort. The algorithm runs until the array elements are sorted and in each iteration two phases occurs – Odd and Even Phases.

pydsa.algorithms.sorting.pigeonhole_sort(arr, key=<function <lambda>>, reverse=False, sorting_algo=<function bubble_sort>)¶

Modified counting sort that sort the range using an underlying algorithm e.g. quicksort. This modified algorithm works for any type of elements.

pydsa.algorithms.sorting.proxmap_sort(arr, key=<function <lambda>>, reverse=False)¶

Proxmap sort is a sorting algorithm that works by partitioning an array of data items, or keys, into a number of “subarrays” (termed buckets, in similar sorts). The name is short for computing a “proximity map,” which indicates for each key K the beginning of a subarray where K will reside in the final sorted order. Keys are placed into each subarray using insertion sort.

pydsa.algorithms.sorting.quicksort(arr, key=<function <lambda>>, reverse=False, pivot_funct=None)¶

A divide-and-conquer algorithm that picks an element as pivot and partitions the given array around the picked pivot.

pydsa.algorithms.sorting.radix_sort(arr, key=<function <lambda>>, reverse=False, order='MSD')¶

Radix sort is a sorting technique that sorts the elements by first grouping the individual digits of the same place value. Then, sort the elements according to their increasing/decreasing order.

pydsa.algorithms.sorting.selection_sort(arr, key=<function <lambda>>, reverse=False)¶

sorts an array by repeatedly finding the minimum/maximum element from unsorted part and putting it at the beginning/end.

pydsa.algorithms.sorting.sleep_sort(arr, key=<function <lambda>>, reverse=False, amplify=1.0)¶

Work by starting a separate task for each item to be sorted, where each task sleeps for an interval corresponding to the item’s sort key, then emits the item.

pydsa.algorithms.sorting.slowsort(arr, key=<function <lambda>>, reverse=False)¶

SlowSort is an example of MultiplyAndSurrender - a worst possible sort algorithm. The algorithm decompose the problem of sorting n numbers in ascending order into:

  1. finding the maximum of those numbers, and

  2. sorting the remaining ones.

Sub-problem (1) can be further decomposed into

(1.1) find the maximum of the first n/2 elements, (1.2) find the maximum of the remaining n/2 elements, and (1.3) find the largest of those two maxima.

pydsa.algorithms.sorting.stooge_sort(arr, key=<function <lambda>>, reverse=False)¶

This algorithm divides the array into two overlapping parts (2/3 each). Then it performs sorting in first 2/3 part and then it performs sorting in last 2/3 part. After that, sorting is done on first 2/3 part to ensure the array is sorted.

pydsa.algorithms.sorting.worstsort(arr, key=<function <lambda>>, reverse=False, recursive_depth=1, sorting_algo=<function bubble_sort>)¶

For k=0, worstsort use bubble sort to sort the array. For any k>0, the algorithm first generates a list of all permutations of the array. Then, it use bubble sort to sort the list of permutations and returns the first element to worstsort(arr, k-1).
